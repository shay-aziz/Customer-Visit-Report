# דו"ח ביקור לקוח – סימטרון 📋

אפליקציית Web לניהול וייצוא דוחות ביקור לקוח של צוות סימטרון ישראל.

---

## תיאור

האפליקציה מאפשרת למהנדסי שטח ואנשי תמיכה למלא דוח ביקור לקוח בצורה נוחה ומהירה, ולייצא אותו כקובץ PDF הכולל את כל פרטי הביקור וחתימת הלקוח.

כל העיבוד מתבצע **באופן מקומי בלבד** בדפדפן — אין שרת, אין בסיס נתונים, ואין שליחת מידע לרשת. הנתונים קיימים אך ורק בזיכרון הדפדפן בזמן מילוי הטופס ומועברים ישירות ל-PDF.

---

## מבנה הטופס

### עמוד 1 – פרטי ביקור
- שם המבקר/המהנדס (בחירה מרשימה)
- תאריך הביקור (בחירה מלוח שנה, פורמט DD/MM/YYYY)
- שם הלקוח / שם העסק
- שם איש הקשר
- טלפון לקוח
- סוג התוכנה (סימטרון / גיבסקאם / פיקוס / מולדקס)

### עמוד 2 – פרטי הפגישה
- שעת הגעה ושעת סיום
- נסיעות – זמן ומרחק (אופציונלי)
- מטרת הביקור (checkboxes עם שדה פירוט לכל אופציה):
  - בדיקת פוסט פרוסוסר
  - מכירות
  - הדרכה
  - תמיכה טכנית
  - התקנה
  - מסירת פוסט פרוסוסר
  - מסירת פלאג
  - אחר
- תיאור הביקור (טקסט חופשי, אופציונלי)
- האם נדרש המשך טיפול (כן/לא + פירוט)
- חתימת הלקוח (כרית חתימה דיגיטלית)

### עמוד סיום
- שמירת PDF מקומית
- שיתוף הקובץ
- חזרה לעריכה

---

## ייצוא PDF

ה-PDF נוצר לוקאלית בדפדפן באמצעות:
- [`html2canvas`](https://html2canvas.hertzen.com/) – המרת תוכן HTML לתמונה
- [`jsPDF`](https://github.com/parallax/jsPDF) – יצירת קובץ PDF מהתמונה

הקובץ נשמר בשם: `דוח-ביקור-[שם לקוח]-[תאריך].pdf`

---

## טכנולוגיות

| טכנולוגיה | שימוש |
|---|---|
| HTML5 | מבנה הטופס |
| CSS3 | עיצוב ו-RTL |
| JavaScript (Vanilla) | לוגיקה, ולידציה, חתימה |
| html2canvas | צילום תוכן לתמונה |
| jsPDF | יצירת PDF |

## הרצה

קובץ סטטי בלבד — אין צורך בשרת או התקנה.

```bash
# פשוט פתח בדפדפן:
open index.html
```

> **הערה:** יש להציב קובץ `logo.png` באותה תיקייה כדי שהלוגו יופיע בכותרת ובPDF.
